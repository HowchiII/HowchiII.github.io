# Hao Qiu Portfolio

Personal portfolio website for GitHub Pages.

## Files to publish

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`

Do not upload `.DS_Store` or resume `.docx` files unless you intentionally want them public.

## GitHub Pages URL

Create a public repository named:

```text
your-github-username.github.io
```

After the files are on the `main` branch, the live site will be:

```text
https://your-github-username.github.io
```
